# Sneghashree Portfolio

A clean and responsive personal portfolio website showcasing my skills, education, experience, projects, and testimonials.

## Tags Used and Descriptions

### 1. **HTML Tags**
| Tag           | Description                                                                 |
|---------------|-----------------------------------------------------------------------------|
| `<!DOCTYPE>`  | Defines the document type and version (HTML5).                             |
| `<html>`      | Root element of the HTML document.                                         |
| `<head>`      | Contains metadata (title, favicon, CSS links).                             |
| `<meta>`      | Specifies metadata (e.g., charset, viewport).                              |
| `<title>`     | Sets the webpage title displayed in the browser tab.                       |
| `<link>`      | Links to external stylesheets or resources like icons.                     |
| `<body>`      | Main content container of the webpage.                                     |
| `<header>`    | Defines the navigation bar at the top of the page.                         |
| `<nav>`       | Navigation links for seamless browsing between sections/pages.             |
| `<ul>`        | Unordered list for navigation links.                                       |
| `<li>`        | List items within the navigation bar.                                      |
| `<a>`         | Anchor tag to link internal/external pages and resources.                  |
| `<main>`      | Wraps the primary content of the webpage.                                  |
| `<section>`   | Divides the page into logical sections like home, about, contact.          |
| `<h1>` - `<h2>` | Headings to structure the webpage content hierarchically.               |
| `<p>`         | Paragraph tag for textual content.                                         |
| `<span>`      | Inline container for styling or adding special effects (e.g., dynamic text). |
| `<img>`       | Embeds images like icons, logos, and project visuals.                      |
| `<form>`      | Defines the contact form for user input.                                   |
| `<input>`     | Input fields for name, email, and other data.                              |
| `<textarea>`  | Multi-line input field for user messages.                                  |
| `<button>`    | Adds clickable buttons (e.g., submit, theme toggle).                      |
| `<footer>`    | Footer section for copyright information.                                  |
| `<script>`    | Includes external JavaScript files for interactivity.                     |

---

### 2. **CSS Selectors**
| Selector              | Description                                                                 |
|-----------------------|-----------------------------------------------------------------------------|
| `body, html`          | Defines global styles like margins, font, and background color.            |
| `.navbar`             | Styles the navigation bar (position, background, shadow).                  |
| `.nav-links`          | Styles the navigation links and layout.                                    |
| `.dark-mode`          | Adds styling for dark mode.                                                |
| `.home`               | Centers and styles the home section.                                       |
| `.social-icons`       | Aligns and styles social media icons.                                       |
| `.form-card`          | Styles the contact form as a clean and interactive card/panel.             |
| `.submit-button`      | Styles the form's submit button with hover effects.                        |
| `#contact-modal`      | Styles the form modal with background and layout.                          |
| `#dynamic-text`       | Applies styles for animated dynamic text in the homepage.                  |

---

### 3. **JavaScript Features**
| Feature               | Description                                                                 |
|-----------------------|-----------------------------------------------------------------------------|
| Dark Mode Toggle      | Allows switching between light and dark themes.                            |
| Dynamic Text          | Adds animated text to highlight changing roles.                            |

---

### 4. **Media Queries**
- Responsive design ensures compatibility with:
  - Tablets (768px).
  - Smartphones (375px).

### Quick Start
1. Clone or download the repository.
2. Open the `index.html` file in a browser.
3. Explore the website, including the interactive contact form.
